# SIMS Hospital User Guide (Staff)

## 1) Login

- Launch **SIMS Hospital** from desktop/start-menu shortcut.
- Sign in with your staff credentials.
- If prompted, change your password before continuing.

## 2) Reception/Billing Workflow

1. Open `Patients` and search by name/phone/MRN/ID proof.
2. Register new patient if not found.
3. Open `OPD Visits` and create a visit:
   - existing/new patient
   - assign doctor
   - consultation fee
4. Track status (`SCHEDULED` -> `IN_PROGRESS` -> `COMPLETED`).
5. Open `Billing` and create invoice with line items.
6. Print invoice in A4 or 80mm thermal format.

## 3) Doctor Workflow

1. Open `Doctor Portal`.
2. Select patient from today queue.
3. Add notes + diagnosis details.
4. Add/update prescription.
5. Mark visit completed.

## 4) Admin Workflow

- `Users`: create, disable, reset passwords
- `Settings`: hospital profile, logo, fees, invoice prefix/sequence

## 5) Billing + Print

- A4 invoice: professional full-page bill
- 80mm invoice: compact thermal print
- Use `Print Bill` button to open print dialog

## 6) Security Practice

- Do not share credentials.
- Reset temporary passwords after staff onboarding.
- Take daily data backups.
